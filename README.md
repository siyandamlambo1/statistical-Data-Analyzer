# 📊 Statistical Calculator

A menu-driven Statistical Calculator developed in Python as part of my programming journey while studying BSc Mathematics & Statistics at North-West University.

## Features

- Add numerical data
- Display data
- Five-number summary
- Calculate the mean
- Calculate variance
- Calculate standard deviation
- Calculate range
- Calculate interquartile range (IQR)
- Search for a number
- Replace numbers
- Delete numbers
- Save data to a text file

## Technologies Used

- Python

## Concepts Practised

- Functions
- Lists
- Loops
- Conditional statements
- File handling
- Exception handling
- Descriptive statistics
